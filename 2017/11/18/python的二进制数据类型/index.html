#bytes,bytearray,memoryview

##bytes
Bytes 不可变的字节序列 ，可以视作整数数组，比如b[0] 返回一个整数

只能用ascii表示（0-255），大于127必须逃逸

##bytearray
bytes 的可变形式。

两种对象都有类似其他序列对象的api
